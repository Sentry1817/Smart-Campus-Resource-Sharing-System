# Smart Campus Resource Sharing & Tracking System

## Project Description
This project is a console-based Java application designed to manage shared campus resources such as projectors, laptops, and lab equipment across multiple departments.

The system prevents double allocation and helps track which department is using which resource.

## Features Implemented
- Store resource details
- Store department details
- Display resources
- Allocate resource
- Release resource
- Prevent double allocation

## Technologies Used
- Java
- Arrays
- Object-Oriented Programming (OOP)

## Project Status
Initial version implemented with resource storage and allocation logic.
